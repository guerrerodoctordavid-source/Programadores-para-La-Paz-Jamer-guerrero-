# Programadores-para-La-Paz-Jamer-guerrero-


Este repositorio contiene las actividades desarrolladas durante el programa Programadores para la Paz.

## Sobre el Programa
Programadores para la Paz es una iniciativa de la Universidad de Cartagena en alianza con el Ministerio de la Igualdad. El programa busca formar desarrolladores de software con enfoque social, brindando herramientas técnicas en desarrollo backend con Node.js, Express, control de versiones con Git/GitHub y buenas prácticas de programación.

## Objetivo
Fortalecer las capacidades tecnológicas de los participantes para que desarrollen soluciones digitales que aporten a la construcción de paz y transformación social en sus territorios.
